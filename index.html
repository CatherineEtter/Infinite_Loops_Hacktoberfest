Infinite Loops Hacktoberfest

Welcome to Infinite Loops Hacktoberfest

This repository has been set up for members of Infinite Loops to learn how to clone repositories, create branches, commit, and make pull requests. This site is for YOU, so feel free to make as many pull requests as you want. As long as you don't try to edit other people's stuff!

Getting Started

This guide will be using Github Desktop

Step 1:

Register to participate in Hacktoberfest here!

Step 2:

Clone this repository using Git, GitHub Desktop, or your favorite editor that's compatible with GitHub

Step 3:

Make a branch

Name your branch

Step 4:

Add your name to the list and commit your changes

Step 5:

Create a pull request

Request a merge

You're done! Once your pull request has been merged, challenge yourself to create your own webpage in the contributors folder and link it from your name!
